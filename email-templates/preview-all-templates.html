🚀 Quick Setup

To enable email sending:

  1. Get a free Brevo API key from app.brevo.com
  2. Create a .env file in the project root
  3. Add your API key:
REACT_APP_BREVO_API_KEY=your_brevo_api_key_here

See email-setup-guide.md for detailed instructions.
